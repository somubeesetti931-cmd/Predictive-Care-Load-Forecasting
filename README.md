# Predictive Forecasting of Care Load & Placement Demand

## Project Overview
This project predicts future healthcare care load and placement demand using machine learning and time-series forecasting techniques.

## Objectives
- Forecast children in HHS care
- Predict discharge demand
- Identify capacity stress
- Provide proactive planning insights

## Technologies Used
- Python
- Pandas
- Scikit-learn
- ARIMA
- Streamlit
- Matplotlib
- Seaborn

## Machine Learning Models
- ARIMA
- Random Forest
- Gradient Boosting

## Features
- Forecast Dashboard
- Model Comparison
- KPI Analytics
- Feature Importance
- Trend Visualization

## Dataset
UAC Program healthcare operational dataset.

## Results
Gradient Boosting and Random Forest achieved strong forecasting performance compared to baseline models.
